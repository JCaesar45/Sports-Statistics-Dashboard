# Sports Statistics Dashboard

## Overview

The **Sports Statistics Dashboard** is an interactive, visually appealing frontend project that showcases real-time sports data visualization techniques. Inspired by the Chicago Bulls theme, this dashboard integrates dynamic data updates, player comparison tools, advanced visualizations like radar charts, and user engagement features such as social sharing and gamification.

Designed to demonstrate frontend skills, responsiveness, accessibility, and innovative visualization techniques, this project is a compelling portfolio piece for aspiring sports industry developers and frontend engineers.

---

## Features

- **Themed UI**: Chicago Bulls-inspired design with red, black, and white color palette.
- **Player Comparison**: Dropdown selectors to compare player stats dynamically.
- **Real-Time Data Simulation**: Periodic updates to simulate live data feeds.
- **Advanced Visualizations**:
  - Bar charts for performance metrics.
  - Radar charts for detailed player stats comparison.
- **Responsive & Accessible**: Mobile-friendly layout with ARIA labels and keyboard navigation.
- **Social Sharing**: Buttons to share insights via Web Share API or fallback alerts.
- **Gamification Elements**:
  - Leaderboards showcasing top players.
  - Placeholder for achievement badges and prediction games.
- **Extendable**: Designed as a scaffold for adding heat maps, heat maps, VR/AR, APIs, and more.

---

## Technologies Used

- **HTML5**: Semantic structure and accessibility.
- **CSS3**: Responsive, themed styling.
- **JavaScript**: Dynamic interactivity and data handling.
- **Chart.js**: Data visualization (bar & radar charts).
- **Web APIs**:
  - Web Share API for social sharing.
  - setInterval for simulating real-time updates.
